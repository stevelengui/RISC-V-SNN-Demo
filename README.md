# RISC-V-SNN-Demo

Deployment of Spiking Neural Networks (SNN) on RISC-V accelerators for IoT devices.  

## Features  
- QEMU-based RISC-V simulation.  
- Power/performance benchmarks vs ARM.  
- PyTorch-to-RISC-V deployment pipeline.
